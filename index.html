Discord Stat Bot

Discord Stat Bot is my personal project. The goal of this project is to create an interactive environment for habit tracking and development by focusing on gamification, accountability, interactivity and data visualization

How does it work?

You can view the code on my Github page

This discord bot acts as the main interface for further communication between the database in Google Sheets and visualization in Tableau.

When started, the bot either transforms an empty google sheet for initial setup, or loads up information from an existing google sheet.

Several skills set by the user are tracked, in the current case these are: Sport, Japanese, Analytics and Mental. When the user advances in any of the areas, they communicate with the bot by using commands such as "$sport" or "$analytics". The bot then indicates that a task has been completed, and awards XP accordingly.
